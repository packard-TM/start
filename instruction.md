# Инстукция по языку Markdown

## Выделение текста.

Для выделения текста курсивом, обрамить текст ОДНОЙ звездочкой:  
*Курсивом.*

Для выделения текста полужирным шрифтом, обрамить текст ДВУМЯ звездочками:   
**Полужирный.**

## Списки.

* Элемент 1
* Элемент 2
* Элемент 3

1. Первый элемент 
2. Второй элемент 
3. Третий элемент
