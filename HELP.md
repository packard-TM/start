# Подсказки по командной строке

Команда смены директории
```sh
cd c:\Folder_name
```

Команда отображения текущей директории для MacOS и Linux
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```
Linux или MacOS:
```sh
ls
```

Удаление файла:
Windovs:
```sh
del file_name
```

Linux или MacOS:
```sh
rm file_name
```

Посмотреть не внесенные изменения
```sh
git diff
```
